# 👥 Proyecto: Listado de Usuarios desde una API REST pública

Este proyecto fue desarrollado para la **Unidad 2 del Módulo 2** del Curso de React JS UTN BA e-learning.

## 🚀 Funcionalidades
- **Consumo Asíncrono:** Uso de `fetch` con `async/await` para obtener datos de la API pública JSONPlaceholder.
- **Gestión de Estados:** Implementación de `useState` para controlar los datos de usuarios, el estado de carga y posibles errores de red.
- **Sincronización:** Uso de `useEffect` para disparar la petición al montar el componente.
- **Arquitectura Modular:** Separación de lógica en `Usuarios.jsx` y representación visual en `UsuarioCard.jsx`.
- **Estilado por Componente:** Organización de archivos CSS independientes para cada componente, mejorando el mantenimiento.
