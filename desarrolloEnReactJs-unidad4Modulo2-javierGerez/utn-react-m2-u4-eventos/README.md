# 📝 Manejo de Eventos en Formularios - UTN React

Este proyecto corresponde a la **Unidad 4 del Módulo 2** del Centro de e-Learning UTN BA. Consiste en un formulario controlado que captura y procesa diversas interacciones del usuario mediante eventos de React.

## 🎯 Objetivos cumplidos
- Captura de eventos `onChange`, `onFocus` y `onBlur` en campos de texto y email.
- Implementación de `onSubmit` con `preventDefault()` para gestionar el envío de datos.
- Detección de teclas específicas (`onKeyDown`) para identificar la tecla Enter en el campo nombre.
- Manejo de eventos de mouse (`onMouseEnter` y `onMouseLeave`) para dinamismo visual en el botón de envío.

### Instrucciones de ejecución:
1. Clonar el repositorio.
2. Instalar dependencias: `npm install`.
3. Ejecutar el proyecto: `npm run dev`.
